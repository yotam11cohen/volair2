## Recommended image dimensions

- Destination cards: 600×800 px (3:4 ratio)
- Hero images: 1920×1080 px (16:9)
- Crew images: 800×600 px (4:3)
- Fleet images: 1200×675 px (16:9)

## Free image sources

- https://unsplash.com (search by city / aircraft)
- https://www.pexels.com
- https://pixabay.com

All images fall back to styled SVG placeholders if the file is missing.
